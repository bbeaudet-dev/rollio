# Multiplayer Components - DEPRECATED

This folder contains multiplayer components that are currently not in use.

## Status

**Deprecated**: These components are archived for future reference but are not currently integrated into the game.

## Components

- `MultiplayerRoom.tsx` - Main multiplayer room component
- `MultiplayerLobby.tsx` - Lobby for multiplayer games
- `MultiplayerGame.tsx` - Multiplayer game component
- `GameHeader.tsx` - Header for multiplayer games
- `HealthCheckStatus.tsx` - Health check status component
- `LiveScoreboard.tsx` - Live scoreboard for multiplayer

## Future Plans

Multiplayer functionality may be re-implemented in the future. These components are preserved as reference.

## Removal

If multiplayer is not planned, these components can be safely removed.
